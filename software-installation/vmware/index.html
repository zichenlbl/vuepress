# Windows 安装 VMware

# 1.1 下载 VMware

进入 WMware 官网：https://www.vmware.com (opens new window)，点击 Resources（资源），点击 Product Downloads (opens new window)（产品下载）。

选择 Desktop & End-User Computing（桌面与终端用户计算）。

选择 VMware Workstation Pro 后面的 Download Product (opens new window)（下载产品）。

选择 GO TO DOWNLOADS (opens new window)（转到下载）

选择 DOWNLOAD NOW（立即下载）。

注册账号登录后进行下载。

下载完后的安装程序。

# 1.2 安装 VMware

双击运行安装程序。
