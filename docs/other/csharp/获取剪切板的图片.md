### 获取剪切板的图片

```csharp
IDataObject mIData = Clipboard.GetDataObject();
if (mIData.GetDataPresent(DataFormats.Bitmap))
{
    Image mImage = (Bitmap)mIData.GetData(DataFormats.Bitmap);
    string mBase64Str = f_ConvertImageToBase64(mImage); //图片转base64
    mBase64Str = "data:image/png;base64," + mBase64Str; //完整base64图片地址
}
```
